# 🚀 Mruknowne9 Code Editor

A powerful, web-based code editor and previewer built with React, Tailwind CSS, and Monaco Editor. This tool allows you to write HTML, CSS, and JavaScript and see the results in real-time.

## ✨ Features

- **Real-time Preview**: See your changes instantly as you type.
- **Monaco Editor**: High-performance code editing experience (the same engine that powers VS Code).
- **Responsive Design**: Works on desktop and mobile.
- **Project Management**: Save and manage your projects (requires login).
- **Publishing**: Share your creations with the world.
- **Export**: Download your code as a standalone `index.html` file.

## 🛠️ Tech Stack

- **Frontend**: React, TypeScript, Tailwind CSS
- **Animations**: Motion (framer-motion)
- **Icons**: Lucide React
- **Backend**: Firebase (Authentication & Firestore)
- **Editor**: @monaco-editor/react

## 🚀 Getting Started

1. **Clone the repository**:
   ```bash
   git clone https://github.com/ismailbanouigu/mruknowne9.git
   ```
2. **Install dependencies**:
   ```bash
   npm install
   ```
3. **Run the development server**:
   ```bash
   npm run dev
   ```
